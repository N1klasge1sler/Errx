# Roadmap

## Phase 1 – CLI Error Explanation
- Capture terminal stderr/stdout
- Normalize error formats
- Generate human-readable explanations

## Phase 2 – Context Awareness
- Source code snippet injection
- Dependency and config awareness
- Improved root-cause detection

## Phase 3 – IDE Integration
- VS Code extension
- Inline error explanations
- Hover-based diagnostics

## Phase 4 – Auto-Fix Agent
- Fix suggestions as diffs
- Apply/reject workflow
- Git integration

## Phase 5 – Pro / Enterprise
- Error memory and analytics
- Team features
- Offline / local-only mode
